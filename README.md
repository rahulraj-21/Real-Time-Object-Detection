# Real-Time Object Detection System

## Overview
Developed a real-time object detection system using Python and computer vision techniques.

## Features
- Real-time object identification using webcam
- Object detection using pre-trained MobileNet SSD model
- Bounding box visualization for detected objects

## Tech Stack
- Python
- OpenCV
- Deep Learning (Pre-trained Models)

## Applications
- Smart surveillance systems
- Automation
- Real-time monitoring

## Run
pip install -r requirements.txt
python object_detection.py
